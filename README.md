# Arduino Nano + HC-05 Bluetooth PCB (KiCad Project)

## Project Overview
This project is a custom PCB designed using KiCad to interface an Arduino Nano with an HC-05 Bluetooth module. The board enables wireless serial communication using UART and is designed for embedded systems learning and prototyping.

---

## Objectives
- Design a functional embedded system PCB using KiCad
- Interface Arduino Nano with HC-05 Bluetooth module
- Understand UART communication (TX/RX)
- Practice schematic design, PCB layout, and routing
- Generate manufacturing-ready Gerber files

---

## Features
- Arduino Nano R3 based design
- HC-05 Bluetooth module interface (UART communication)
- Power and ground distribution
- Status LED indicators
- Push button input support
- Clean PCB layout for prototyping

---

## Circuit Description
- Arduino Nano communicates with HC-05 using UART:
  - TX → RX
  - RX → TX
- Powered via 5V and GND
- Optional LED indicators for system status
